About Me

I'm Debapriyo Pal Chowdhury, a motivated Computer Science student at JIS School of Polytechnic, pursuing a Diploma in Computer Science & Technology. I specialize in full-stack web development, combining frontend technologies like HTML, CSS, JavaScript, and Bootstrap with backend frameworks like Django and MySQL database management.

During my industrial training at Euphoria Genex, Kolkata, I gained hands-on experience developing dynamic web applications, implementing user authentication, admin dashboards, product listings, and order management systems using Django views and models. I also connected applications to MySQL databases via XAMPP for efficient data storage.

I am proficient in Python programming, front-end web design, database integration, and version control using Git. My projects showcase my ability to build responsive, interactive, and user-friendly web applications that are both performant and visually appealing.

Beyond coding, I actively explore new web development trends, experiment with UI/UX design, and contribute to open-source projects. I am passionate about learning and applying modern technologies to deliver impactful and scalable digital solutions.
